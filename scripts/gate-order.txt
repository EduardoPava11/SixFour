# SixFour gate order — the sequence `scripts/s4.sh all` runs, one verb per line.
# Rationale for the ordering (each step depends on artifacts the previous produced):
#   codegen → regenerates SixFour/Generated/* + trainer/generated/* from the spec.
#   verify  → cabal test: proves every Spec law + per-kernel s4_* law (the source of truth).
#   native  → zig build test (host fixtures cross-check Haskell goldens) + device static lib.
#   lint    → lint-grid.sh; READS Generated/, so must run AFTER codegen.
#   gen     → xcodegen + pbxproj patch; produces the .xcodeproj Swift will build.
#   build   → xcodebuild; Swift imports the generated contracts, so it runs LAST.

verify
native
