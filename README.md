# SixFour

iOS 26 camera app that turns a 64-frame burst into a **64 × 64 × 64-frame
animated GIF**, with each frame quantised against a per-frame 256-colour OKLab
palette engineered for maximum LAB diversity. Capture, palette generation,
dithering, and GIF encoding all run on device.

> **Orientation.** For the project narrative see [`docs/SIXFOUR-VISION.md`](docs/SIXFOUR-VISION.md);
> for current state (what's built / designed / missing) see
> [`docs/SIXFOUR-ARCHITECTURE-MAP.md`](docs/SIXFOUR-ARCHITECTURE-MAP.md). Superseded design docs
> live in [`docs/archive/`](docs/archive/) and [`spec/archive/`](spec/archive/).

> **Engineering contract:** Haskell-verified, dependency-free, hand-written.
> The shipped iOS app has **zero third-party dependencies**; MLX (M1) is a
> Mac-side training tool only. See [`CLAUDE.md`](CLAUDE.md) for the full
> three-tier dependency contract.

A successor "look-network" (look-NN) is being designed in parallel: it
collapses the 64 per-frame palettes into a single **global 256-colour palette
organised as 128 σ-balanced complementary pairs**, plus per-frame index maps,
so the 64³ voxel mass renders with one shared palette. The look-NN's
architectural commitments live in `spec/` (contract-first Haskell) and are
verified at the tensor level in `spec/src/SixFour/Spec/{Quad4Fit,SigmaPairHead}.hs`
before any inhabitant ships.

## Layout

| Path | Purpose |
|---|---|
| `SixFour/` | iOS 26 app source — capture pipeline, Metal shaders, GIF encoder, UI |
| `SixFourTests/` | Swift unit tests |
| `spec/` | Haskell algebraic spec — the source of truth for shapes, laws, codegen |
| `studio/` | Rust proprietary workspace — analysis-core, look-nn forward net (cargo-deny-gated) |
| `trainer/` | Python (MLX) trainer — consumes generated Python contracts |
| `scripts/` | Repo housekeeping |

The Haskell spec emits Swift, Python, and Rust contracts so the three runtime
implementations (iOS app, MLX trainer, Rust forward net) cannot drift from the
algebraic source of truth.

## Build

### iOS app

Project is generated by [xcodegen](https://github.com/yonaskolb/XcodeGen):

```bash
xcodegen
open SixFour.xcodeproj
```

Builds for iPhone 17 Pro simulator (Xcode 26.2, iOS 26, Swift 6.2). For
device builds, set the team in `project.yml`.

### Spec + codegen (Haskell)

```bash
cd spec
cabal build
cabal test           # 191 properties; ≈1 min
cabal run spec-codegen
```

`spec-codegen` writes generated files to:
- `SixFour/Generated/` (Swift contracts: stages, net, hybrid, significance)
- `trainer/generated/` (Python: stages, net_shape)
- `studio/look-nn/src/generated/contract.rs` (Rust dimensional constants)
- `SixFour/Resources/stbn3d-8.bin` (STBN3D dither mask, 8³, tiled to 64³ at runtime)

Three accompanying executables ship in the cabal:
- `cabal run spec-tui` — terminal animation of L1…L9 through a CyclicStack
- `cabal run spec-gif` — renders contract proofs as animated GIFs
- `cabal run spec-gen` — generates full 64³ GIFs as test data (byte-faithful port of `GIFEncoder.swift`)

### Studio (Rust)

```bash
cd studio
cargo deny check licenses        # permissive-license gate
cargo build -p analysis-core -p look-nn
```

The `look-nn` crate is the pure-Rust v1 forward net + 1+1-ES search over the
encoder genome (per `spec/archive/COMPETITION.md` — note: the 1+1-ES genome was
superseded by the continuous pivot / `Spec.Preference`; see that archived doc's banner).
The autodiff (`burn`) backend is the deferred v2 upgrade pending dep-tree license vetting.

### Trainer (Python)

```bash
cd trainer
uv sync
python train_metric.py
```

Reads `generated/stages.py` and `generated/net_shape.py` — the shapes that
must match the on-device contract.

## Architectural pillars

1. **Contract-first.** Every dimension and law is defined in `spec/`
   (Haskell, QuickCheck-verified) before any runtime implementation. Codegen
   pins Swift / Python / Rust to the same constants, so drift fails the build.

2. **Substrate is OKLab + Wasserstein-2 (Bures) geometry on Gaussians.**
   Each frame's 256-cluster palette is consumed as a Gaussian mixture
   (`Spec.GMM`, μ + Σ + weight per cluster). Palette collapse is the W₂
   barycenter on Gaussians (`Spec.Bures`), with the k-means floor as the
   Σ → 0 limit. This is the Wasserstein-style-transfer lineage applied to
   temporal palette collapse — see `spec/LOOK_NN.md`.

3. **σ-symmetry is structural, not learned.** The σ involution
   `σ(L, a, b) = (L, −a, −b)` is the chromatic complement. The look-NN's
   decoder is `Spec.SigmaPairHead` — a depth-7 binary Haar pyramid on **128**
   OKLab generators `c_i`, with the 256-leaf palette defined as
   `[c_0, σ(c_0), c_1, σ(c_1), …]`. The σ-pair structure is algebraic; the
   palette is σ-symmetric for *any* genome, not just well-trained ones.

4. **Tensor-verified architectural claims.** Type-class composition is
   shape-only. The look-NN's representational power is verified by computing
   the **design matrix `B`**, its **rank**, and the **orthogonal projection
   residual** on synthetic targets. SigmaPairHead's design matrix is
   `768 × 384`, full rank, with **zero residual on σ-symmetric palettes** and
   **~9 % residual on random palettes** (the irreducible σ-asymmetric
   content). The earlier Quad4 design failed this test — same residual on
   both, no preferential σ-alignment — and is documented as a rejected
   alternative.

5. **No-stubs discipline.** Modules ship complete or not at all. The spec
   library is the only place new dimensions or laws are introduced; runtime
   code consumes generated contracts.

## Status

- **Spec library: 191 QuickCheck properties green** (`cabal test`).
- 16 layer-law report passes at the production 64³ shape across three seeds
  (`Properties.Scale`).
- Cross-language golden checks: Haskell ↔ Rust on Bures, GMM, Diversity,
  Coverage, and Cyclic descriptor.
- Tensor residual checks: Quad4 rejected; SigmaPairHead adopted; PairTree
  retained as the universal-fit baseline.
- iOS app: capture → per-frame Wu+KM palette + GPU blue-noise dither → GIF,
  on-device verified.
- Look-NN: deterministic forward net wired (`studio/look-nn`); gradient
  trainer (`burn` v2) deferred; on-device weights not yet shipped.
